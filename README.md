# EDA-80-Cereals

* Performing exploratory data analysis (EDA) and data visualizations on a small dataset, `80 Cereals`.

* The dataset can be found [here](https://www.kaggle.com/datasets/crawford/80-cereals).

* Tools used:

	* Python
	* Jupyter notebooks
	* Numpy
	* Pandas
	* Matplotlib
	* Seaborn
